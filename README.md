# Predictive Analytics Using Historical Data

## Overview
This project demonstrates predictive analytics using historical data. It performs:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Linear Regression model training
- Model evaluation
- Prediction visualization

## Project Structure
```
Predictive_Analytics_Project/
├── data/
│   └── sample_sales_data.csv
├── src/
│   ├── train.py
│   └── visualize.py
├── models/
├── requirements.txt
└── README.md
```

## Installation
```bash
pip install -r requirements.txt
python src/train.py
```

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Future Scope
Replace the sample dataset with a real dataset and extend the project using ARIMA, Prophet, or LSTM.
